# 回溯算法
## 集合
题号[77. 组合](https://leetcode.cn/problems/combinations/)

### 思考

1. 回溯法三部曲
	1. 递归函数的返回值以及参数
	2. 回溯函数终止条件
	3. 单层搜索的过程

##### 方法一 

```ts
function combine(n: number, k: number): number[][] {
    let path = [],result = []
    function backTraking(n,k,startIndex){
        if(path.length===k){  // 终止条件
            result.push([...path])
            return 
        }
        for(let i=startIndex;i<=n;i++){
            path.push(i)
            backTraking(n,k,i+1)
            path.pop()  // 回溯
        }
    }
    backTraking(n,k,1)
    return result
};
```

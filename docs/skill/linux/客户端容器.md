## 客户端容器

### 浏览器架构演进

1. 单进程架构
2. 多进程架构（现代浏览器多为这个）
3. 面向服务架构

### 首屏优化

1. 压缩、分包、删除无用代码
2. 静态资源分离
3. js脚本非阻塞加载（js引擎和render是互斥的）
4. 缓存策略
5. SSR
6. 预置loading、骨架屏

### 渲染优化

1. GPU加速（transform也会GPU加速）
2. 减少回流、重绘
3. 离屏渲染
4. 懒加载

### JS优化

1. 防止内存泄漏
2. 循环尽早break
3. 合理使用闭包
4. 减少DOM访问
5. 防抖、节流
6. Web Workers
